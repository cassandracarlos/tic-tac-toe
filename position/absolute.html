Absolute Box

Container (.container):

This div has position: relative, which means it is positioned according to the normal flow of the document, but it acts as a positioned ancestor for any absolutely positioned children. It has a set width and height and a light grey background.

Absolute (.absolute):

This div has position: absolute, meaning it will be taken out of the normal document flow and positioned relative to its closest positioned ancestor, which is .container in this case. It is positioned 20 pixels from the top and 50 pixels from the left of the .container element. It has a set width and height, a light coral background, and a black border.
